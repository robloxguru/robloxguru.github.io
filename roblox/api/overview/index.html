Overview

IMPORTANT: Please note that this is project made by kablankooo. Issues may arise within the codebase due to discord.js updates or ROBLOX API changes.
This API service must be ran within your own system. You may host it yourself either through your own device, you will need to port forward, or use an online hosting service such as Hostinger or bot-hosting.net. This API uses Lua within ROBLOX and node JS within modules. You will need to run certain commands and install the correct languages into your service / hoster. You may find this in the tutorial.
